相关概念

  • DB:

  • 数据库，保存一组有组织的数据的容器

  • DBMS:

  • 数据库管理系统，又称为数据库软件，用于管理DB中的数据

  • SQL:

  • 结构化查询语言，用于和DBMS通信的语言

语言规范

  • 不区分大小写，但建议关键字大写，表名、列名小写
  • 每条命令最好用分号结尾
  • 每条命令根据需要，可以缩进或换行，不影响命令执行
  • 注释

    单行注释：#注释文字
    单行注释：--注释文字
    多行注释：/*注释文字*/

常用命令

  • 查看版本
    select version();

  • 打开数据库test
    use test

    若显示Database changed; 表示打开成功

  • 显示本库的表
    show tables;

    显示Empty set表示库为空

  • 打开指定库的表（换库）（库名mysql）
    show tables from mysql

  • 查看在哪个库内
    select database();

  • 建表a

    显示Query OK即成功

  • 显示a表结构
    desc a

  • 显示数据
    select * from a

  • 添加数据
    insert into a (id,name) values(1,'john')

  • 修改数据
    update a set name='A' where id=1;

  • 删除数据
    delete from a where id=1;
    delete from a where name='john';

  • 建库
    create database a;

  • 删库
    drop database a;

  • 删表
    drop table a;

  • 查询

    普通查找

    从某张表对某一列进行查找

    select 列名称 from 表名称;

    从某张表对所有字段进行查找

    select * from 表名称;

    条件限制查找

    select * from 表名 where=限制条件;

  • 更新

    update 表名称 set 列名称=新值 where 列名称=某值;
